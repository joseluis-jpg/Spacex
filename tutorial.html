¿Cómo jugar a Spacex?

En *Spacex*, el objetivo es eliminar las naves enemigas que se acercan. Tienes tres vidas para completar la misión, así que ¡ten cuidado con las naves enemigas kamikaze!

Controles

  • Movimiento: Usa las teclas de dirección o el mouse para mover tu nave.
  • Disparar: Pulsa la barra espaciadora para disparar.
  • Evitar colisiones: Ten cuidado de no chocar con las naves enemigas, ya que perderás una vida.

Reglas del juego

  • El objetivo principal es destruir las naves enemigas para ganar puntos.
  • Tienes tres vidas. Cada vez que colides con una nave enemiga pierdes una vida.
  • Cuando eliminas naves, obtienes puntos que se suman a tu marcador.
  • Si pierdes todas tus vidas, el juego termina.
